# 單字資料說明

## 資料結構

單字資料存放在 `assets/data/words.json`，每個單字包含以下欄位：

- `word`: 英文單字（基本形式）
- `translation`: 中文翻譯（待補充）
- `partOfSpeech`: 詞性
- `exampleEn`: 英文例句
- `exampleZh`: 中文例句
- `cambridgeUrl`: 劍橋字典連結
- `level`: 級別（1-6）
- `audioUrl`: 音檔連結（可選）

## 如何添加完整單字列表

1. 將完整的單字列表（包含所有六個級別）保存到 `scripts/words_input.txt`

2. 執行腳本生成 JSON：
   ```bash
   cd scripts
   python create_complete_words.py
   ```

3. 腳本會生成 `assets/data/words.json`，包含所有單字的基本結構

4. 翻譯欄位（translation）需要後續從劍橋字典獲取

## 獲取中文翻譯

由於單字數量龐大（數千個），建議：

1. **批量處理**：使用腳本分批查詢劍橋字典 API（如果有）
2. **手動補充**：對於常用單字，手動從劍橋字典網站獲取翻譯
3. **使用現有資源**：查找是否有現成的學測單字翻譯資料庫

## 級別說明

- **第一級**：基礎單字（約 1000+ 個）
- **第二級**：進階單字
- **第三級**：中級單字
- **第四級**：中高級單字
- **第五級**：高級單字
- **第六級**：最高級單字

## 注意事項

- 確保 JSON 文件格式正確
- 翻譯欄位不能為空（至少要有基本翻譯）
- 劍橋字典連結會自動生成，但需要確保單字拼寫正確
