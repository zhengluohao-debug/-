# 如何添加完整的六個級別單字列表

## 問題說明

目前 `words_input.txt` 只有第一級的單字，而 `sample_words.json` 只有 5 個示例單字（都是第 4 級）。

## 解決方法

### 方法 1：使用 Python 腳本（推薦）

1. **將完整的六個級別單字列表保存到 `scripts/words_input.txt`**
   - 確保文件包含「第一級」到「第六級」的所有單字
   - 格式應該像這樣：
     ```
     第一級
     
     a/an art.
     ability n.
     ...
     
     第二級
     
     absence n.
     ...
     ```

2. **安裝 Python（如果還沒安裝）**
   - 下載：https://www.python.org/downloads/
   - 安裝時勾選「Add Python to PATH」

3. **運行腳本生成 JSON**
   ```bash
   cd scripts
   python create_complete_words.py
   ```

4. **腳本會生成 `assets/data/words.json`**，包含所有單字的基本結構

### 方法 2：手動創建 JSON（不推薦，因為單字太多）

如果 Python 無法使用，可以手動編輯 JSON，但這會非常耗時。

## 注意事項

- 翻譯欄位（translation）目前為空，需要後續從劍橋字典獲取
- 確保 JSON 文件格式正確
- 每個單字都需要有 `level` 欄位（1-6）

## 級別說明

- **第一級**：基礎單字（約 1000+ 個）
- **第二級**：進階單字
- **第三級**：中級單字
- **第四級**：中高級單字
- **第五級**：高級單字
- **第六級**：最高級單字

## 獲取中文翻譯

由於單字數量龐大（數千個），建議：
1. 使用劍橋字典網站手動查詢常用單字
2. 查找是否有現成的學測單字翻譯資料庫
3. 分批處理，先處理常用單字
